# Batch Export FBX for Blender

批量导出FBX插件，支持物体位置归零和原地导出两种模式。

## 🚀 功能特点

- **批量导出**: 将选中的多个物体分别导出为独立的FBX文件
- **自动命名**: 导出的FBX文件将自动以物体名称命名
- **双模式导出**:
  - **位置归零**: 默认情况下，导出时会将物体移动到原点位置
  - **原地导出**: 按住Shift键点击导出按钮时，物体将保持在原位置导出

## 📖 使用方法

### 基本操作
1. 在Blender中选择要导出的物体
2. 在3D视图右侧的"CC"面板中找到"Batch FBX Export"工具
3. 设置导出目录路径
4. 点击"Batch Export FBX"按钮

### 导出模式
- **普通点击**: 物体先移动到原点，然后导出
- **按住Shift点击**: 物体保持在原位置导出

## 🛠 安装方法

1. 下载整个插件文件夹
2. 将文件夹复制到Blender的插件目录（通常是 `scripts/addons/`）
3. 在Blender中：编辑 → 偏好设置 → 插件 → 搜索"Batch Export FBX"
4. 启用插件
5. 插件将在3D视图的"CC"面板中显示

## ⚠️ 注意事项

- 确保导出目录存在或有写入权限
- 导出时会临时修改物体位置，导出后会自动恢复
- 使用Shift+点击可以避免位置变化，适合需要保持世界坐标的场景
- 插件支持Blender 2.80及以上版本
